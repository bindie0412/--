# Study Sanctuary

A minimalist, high-end digital sanctuary for deep study and focus.

## Features
- **Aura Zen UI:** Dark mode designed to reduce eye strain and promote focus.
- **Pomodoro Timer:** Centralized timer with circular progress.
- **Focus Mode (AOD):** A minimalist desk-clock mode for distraction-free sessions.
- **Competition Stats:** Compare your current focus performance with your past self.
- **White Noise Mixer:** Blend atmospheric sounds including Neural Flow (Alpha) waves.
- **Integrated To-Do & Calendar:** Keep track of tasks and sessions without leaving the app.
- **Community Status:** See how others are progressing in real-time.

## Tech Stack
- **Frontend:** Next.js, React, Tailwind CSS, Recharts, Lucide Icons.
- **Backend:** FastAPI (Python).

## Getting Started

### Frontend
1. cd frontend
2. npm install
3. npm run dev

### Backend
1. cd backend
2. pip install -r requirements.txt
3. uvicorn app.main:app --reload
